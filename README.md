# ParkerBecomeProbe

## Clone the repo
### `git clone https://github.com/aymen-ghemam/space-apps-challenge.git`

## Install Dependencies
In the project directory, you can run:
### `npm install`

## Run the App
In the project directory, you can run:
### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
